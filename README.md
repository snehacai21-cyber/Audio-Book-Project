## BookMind – AI Book Summarizer

### Features
- Summarizes large books (500+ pages)
- Supports TXT, PDF, DOCX
- Generates text + audio summary
- Uses T5 Transformer model

### Tech Stack
- Python (Flask)
- HuggingFace Transformers
- HTML/CSS
- TTS

### How to Run
pip install -r requirements.txt  
python app.py
